# rich-get-richer
experiment should the rich get richer?

#requerments
1. monogoDB
2. nodeJS
3. bower 

#run node server

```bash
cd songs_Nodejs.
npm install
node index.js.
```

#install the client side
```bash
bower install
```

##config.
in `js/controllers/pagesCtrl.js` change url to your server URL

and you are read to start :)
