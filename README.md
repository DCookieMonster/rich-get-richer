# rich-get-richer
experiment should the rich get richer?

##requerments
1. monogoDB
2. nodeJS
3. bower 

##run node server

```bash
cd Nodejs_logger
npm install
node index.js
```

##install the client side
```bash
bower install
```

###config.
In `js/controllers/pagesCtrl.js` change url to your server URL

and you are ready to start :)
